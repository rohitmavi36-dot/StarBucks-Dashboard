# StarBucks-Dashboard
☕ Starbucks Beverage Analytics Dashboard

📌 Overview

This project is an interactive Power BI dashboard that provides insights into Starbucks beverage data. It analyzes key metrics such as calories, sugar content, caffeine levels, beverage categories, and global presence through dynamic visualizations.

🚀 Features
Total Beverage Analysis
Average Calories, Sugar, and Caffeine Metrics
Beverage Category Distribution
Category-wise Calorie Analysis
Category-wise Caffeine Analysis
Top 5 Highest Caffeinated Beverages
Starbucks Global Presence Visualization
Interactive Filters and Slicers

🛠️ Tools & Technologies
Power BI
DAX
Data Modeling
Data Visualization

📊 Dashboard Insights
Track nutritional trends across beverage categories.
Compare caffeine content among different drinks.
Identify high-calorie and high-sugar beverages.
Explore beverage distribution and category performance.

🎯 Objective
The goal of this project is to transform Starbucks beverage data into meaningful insights using interactive dashboards, enabling better understanding of product nutrition and category trends.
